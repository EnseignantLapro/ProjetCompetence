🔄 Test de Conflit

Pour tester le remplacement automatique des tokens :

  1. Connectez-vous d'abord avec un token élève
  2. Puis cliquez sur un lien enseignant - le token élève devrait être automatiquement supprimé
  3. Et vice-versa

🔍 Vérification localStorage

Ouvrez les outils de développement (F12) et allez dans l'onglet "Application" > "Local Storage" pour voir les tokens stockés.

Vous devriez voir que quand vous passez d'un type de token à l'autre, l'ancien est automatiquement supprimé.
